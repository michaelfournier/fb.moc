== Description ==


A plugin to Remove selected stuff/junk from Wordpress Head meta section like extra feed links, wlmanifest link, next/previous post link, wp_generator etc.
Using this plugin you can select which meta to remove from your wordPress and which are to keep. 

List of Metas you can remove using this plugin:

1. WP XHTML generator Showing Wp Version

2. RSD Link

3. Links to the general feeds (e.g. posts and comments)

4. Links to the extra feeds such as category feeds

5. Link to the Windows Live Writer manifest

6. Index link

7. Relational links for adjacent posts

8. Previous posts Link

9. Start posts Link
10. Alkivia Framework Meta Removal

For Support visit [Plugin Support Page](http://www.designzzz.com/wordpress/meta-head-cleaner-free-plugin/ "Wordpress Head Section Cleaning Plugin")

== Installation ==


1. Upload `designzzz-clean-head.zip` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

And we are done :}
